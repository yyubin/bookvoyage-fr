# Bookvoyage

책 리뷰를 공유하고 추천을 나누는 커뮤니티 UI 프로토타입입니다.  
Next.js App Router 기반으로 피드, 책/리뷰 상세, 프로필, 검색, 리뷰 작성 흐름을 구성했습니다.

## 주요 화면
- 홈 피드/추천, 리뷰 카드, 태그 검색
- 리뷰 상세 + 댓글 모달, 이모지 리액션
- 책 상세 + 리뷰 모아보기
- 프로필(동적) + 리뷰/북마크/읽는 중 전개 페이지
- 로그인/회원가입(OAuth UI), 리뷰 작성 폼

## 데이터/서비스 구조
- 예제 데이터 + 서비스 레이어 fallback
- 커서 기반 페이지네이션 구조
- 검색 서비스(결과 없을 시 empty 처리)

## 디렉토리 참고
- `src/app` : 페이지/컴포넌트
- `src/app/data` : 예제 데이터
- `src/app/services` : 서비스 레이어(fallback 포함)
- `src/app/types` : 타입 정의
